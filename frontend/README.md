# MERN E-Commerce Project

A modern e-commerce web application built with the **MERN stack** (MongoDB, Express, React, Node.js) using **TypeScript** and **Redux Toolkit** for state management.

## Tech Stack
**Frontend:** React + TypeScript + Vite + Tailwind CSS + Redux Toolkit  
**Backend:** Express.js + TypeScript + MongoDB + Mongoose  
**Auth:** JWT with Email OTP verification  
**Other:** Axios, Bcrypt, Nodemailer

## Current Features
- User registration & login with OTP email verification  
- JWT-based authentication
- User profile & protected routes setup with authentication and authorization services

## Upcoming
- Product & Category management  
- Admin dashboard  
- Orders, Payments, and Cart system
